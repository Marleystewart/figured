# Athlink

Athlink is a dark-mode basketball recruiting web app concept for athletes pursuing overseas, pro, semi-pro, college, agent, and scout opportunities.

## Run locally

This is a static HTML/CSS/JavaScript project. No package install is required.

```bash
python3 -m http.server 4174
```

Then open:

```text
http://127.0.0.1:4174/
```

## Files

- `index.html` - app structure and screens
- `styles.css` - dark sports-tech interface styling
- `script.js` - tab navigation and simple interactions
- `assets/basketball-highlight.png` - basketball action image asset
